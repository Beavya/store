## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/Beavya/store.git
cd store
```

### 2. Установка зависимостей
```bash
npm install
```

### 4. Запуск в режиме разработки
```bash
npm run serve
```
После запуска откройте браузер по адресу: `http://localhost:8080`

## Сборка проекта

### Для продакшена
